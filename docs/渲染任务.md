# 渲染任务

## 概要
此项目为设计软件 `图册` 渲染任务窗口，用于查询任务及其渲染状态、效果图调节下载等功能。

* ___该项目也是内嵌页面项目，但由于不在embed仓库中，所以没有放在内嵌页目录下___
