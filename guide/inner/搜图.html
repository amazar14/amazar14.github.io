#2. 设计软件通信

#2.1 点击资源传递已选中图片信息给设计软件

  • 搜图页面，点击鼠标左键选中图片后，将选中的图片id等信息发给设计软件
点击资源传递已选中图片资源给设计软件
1// send to homedesign
2let params = {
3  code: "SEARCH_IMAGE_SET_RESOURCE",
4  param: {
5    name: val.Name, // 名称
6    Id: val.Id, // 当前选择的资源的id
7    Catalog: val.Catalog, // 目录id组成的路径
8    anoterName: val.AnotherName, // 别名
9    FiList: val.FiList, // 包含资源所有的xml、json、jpg、cst、hdf等类型的文件地址（FileUrl）、文件类型（Filetype）、文件后缀（FileEtag）数组
10    RealId: val.RealId, //  资源真正的ID  RealId
11    Source: val.Source, // 资源的类型， 0 代表原始资源， 1代表映射资源， 2代表引用资源
12  }
13}

#2.2 页面点击【返回】关闭搜图页面

  • 点击页【返回】按钮，调用指令，关闭搜图页面
关闭
1// send to homedesign
2  let params = {
3    code: "SEARCH_IMAGE_CLOSE",  // websocket交互指令
4}
