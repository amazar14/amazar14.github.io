#资源库管理

#1. 概述

设计软件资源库管理

#2. 地址

正式地址: https://help.ihomecloud.com.cn/tool/dealer/managerEntLib?tenantCode=xxx&tokenId=xxx&port=1001

测试地址: https://uat-help.ihomecloud.com.cn/tool/dealer/managerEntLib?tenantCode=xxx&tokenId=xxx&port=1001

参数说明
tenantCode租户code
tokenId当前用户的tokenId
port端口号
action操作类型(见下表)

操作类型: 默认为空，表示打开资源库管理页面。其他值见下表：

action说明
SELECT_CATALOG选择目录

#3. 设计软件通信

#3.1 复制资源到指定目录

选择目录后会向软件发送如下消息

send to homedesign
{
	code: "SELECT_CATALOG",
	params: {
		catalogId: "", // 目录ID
	}
}

#3.2 管理企业资源

资源管理页面关闭后，软件刷新左侧资源。

  • 每次操作后会向软件发送如下消息
send to homedesign
{
	code: "UPDATE_ENT_LIB",
	params: {}
}
