#2. 创建配置

只有 DefCatalog默认配置库可以创建配置，其他配置库只能导入配置

从已有的配置中选择配置，配置名称和ID不需要填写，会从选择的配置中带出

#3. 导入配置

将其他库的配置导入到当前库，（此操作为配置的复制，导入后的修改不影响原来的配置）

#材质业务表

#1. 表达式规则

渠道组表示：在一个或多个渠道下，在什么条件下结果是什么值。例：{渠道 # 值?条件 ; 值?条件}

  1. 一个或多个渠道值为一个渠道组，
  2. {}: 包裹每个渠道组。
  3. @ : 连接渠道组中多个渠道值。
  4. # : 分割渠道与值?条件
  5. ; : 连接多个值?条件
  6. ? : 值?条件

  • 渠道组

    '{渠道值@渠道值2#值?条件;值?条件}{剩余渠道#值?条件}'
    #example
    // 在渠道79中和61，条件为condition时，值为value，
// 其他渠道下值为value1
'{product_channel_79@product_channel_61#value?condition}{剩余渠道#value1}'

  • 全部渠道

    只有值的情况，仅字符串

    'value'

    多个条件的情况

    '{剩余渠道#值?条件;值?条件}'

#渲染水印

#无配置库时

设计软件会下载 channelCode.xml配置文件 + logo图片 的压缩包，xml文件及logo图根据后台管理中租户渠道配置生成。 设计软件根据xml文件中的 渠道ID + LOGO名称 获取logo图。

channelCode.xml 配置文件
<HDChannelLogoInfo>
  <Item ID="210" NAME="miform木门miform零售" LOGO="miform"/>
  <Item ID="167" NAME="欧派橱柜集成厨房" LOGO="欧派"/>
  <Item ID="70" NAME="欧铂丽木门整装" LOGO="欧铂丽"/>
</HDChannelLogoInfo>

#改为配置库后

  1. 前端提供json文件，供后端生成压缩包zip
  1. 配置文件说明

配置库中保存的配置文件不会直接发送给软件使用，经过转换后两端文件会有差异。

  • 软件中配置文件，至少包含ID、NAME、LOGO字段
channelCode.json
{
  ID: "210",
  NAME: "miform木门miform零售",
  LOGO: "miform"
}
  • 配置库中配置文件

ID、NAME、LOGO对应xml中相同字段，url地址为四个清晰度的图片地址，下划线 _ 开头的字段只在配置库页面中使用

waterMark.json
[
  {
    "ID": "210",                            // 渠道值末尾数字
    "NAME": "miform木门miform零售",          // 品牌 + 品类 + 渠道名  
    "LOGO": "miform",                       // 水印名称 
    "logoUrl4K": "https://uatcdn.jiajutech.com/jiaju/ability/image/995b1e03-91a8-43ff-976f-c3471bd2bfe7.png",
    "logoUrlFHD": "https://uatcdn.jiajutech.com/jiaju/ability/image/181b0d3a-c7ff-4d9c-949d-3c963071b348.png",
    "logoUrlHD": "https://uatcdn.jiajutech.com/jiaju/ability/image/739256ec-a1c8-4e7a-bdb0-8308373260c2.png",
    "logoUrlSD": "https://uatcdn.jiajutech.com/jiaju/ability/image/ec5095dc-4d56-42c2-9c8a-abab453c0a2a.png",
    // 以下字段只在配置库中用到
    "_brandLogoCode": "",                   // 水印code
    "_brand": "miform",                     // 品牌
    "_category": "衣柜",                    // 品类
    "_channelName": "整装大家居",            // 渠道名 
    "_channelCode": "product_channel_210"   // 渠道值
  }, 
  ...
]
