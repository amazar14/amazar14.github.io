#登录

#1. 获取登录信息

用户登录后，用户信息会储存到本地，在每一个 .vue、.js 文件中都可获取。可以直接使用JSON.parse(sessionStorage.getItem('login')) 或使用工具函数获取（建议）

示例

.js文件
1import { getUserInfo } from "@/util/index.js"
2  
3  // 工具函数
4  const userInfo = getUserInfo()
5  // 如果不想用工具函数
6  const userInfo = JSON.parse(sessionStorage.getItem("login"))
7  console.log(userInfo)
8
9  // 浏览器控制台会显示 userinfo信息，userInfo.user_info获取信息
10  // {
11  //   access_token: '6e79f7be-85a1-464e-9a5a-872c80eb4ba8',
12  //   ... ,
13  //   user_info: {
14  //     username: 'xxx',
15  //     authorities: [],
16  //     ...
17  //   }
18  // }
.vue文件
1<template>
2  ...
3</template>  
4<script>
5  import { getUserInfo } from "@/util/index.js"
6  export default{
7    data()=>{},
8    created(){
9      getUserInfo()
10    },
11    mounted(){
12      getUserInfo()
13    },
14    methods:{
15      // 你的函数
16      fn(){
17        getUserInfo()
18      }
19    }
20  }
21</script>
22<style></style>
