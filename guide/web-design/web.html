#Web端相关说明

#1. api

所有api接口由web端定义，web端暴露 service 对象，由软件调用。

api示例
1// web暴露出请求实例
2  service = {
3    // 请求默认为异步
4    async getFileUrlByZyId():Promise<any>{
5      // send httpRequest
6      return promise
7    },
8    // 提供对应请求的同步方法 (方法名后拼接 _sync)
9    getFileUrlByZyId_sync():any{
10      // send httpRequest
11      return result
12    }
13  }
14
15// 设计软件端使用 service
16function doSomething(){
17  // 异步await接收
18  const result = await service.getFileUrlByZyId()
19}
20
21const result1 = service.getFileUrlByZyId_sync()
⚠警告

虽然提供了对应的同步方法，但强烈不建议使用同步发送请求。

#1.1 api定义

src/api/*.ts 文件夹下文件对应正常的api请求，对接口的 缓存 及 并发限制 在proxy文件夹下对应 src/api/*.ts 的每个文件配置

示例
import { GetSomething } from "../api/*.ts"

type Option = {
  cache?: boolean
  limit?: boolean
  sync?: boolean
  transform: (any) => any,
  cacheKey: (any) => string
}

export default [
  {
    name: "GetSomething_Cache_Limit",
    handler: GetSomething,
    option: {
      cache: true,
      limit: true,
      sync: false,
      transform: (res)=>{
        if (res.success){
          // 成功才会缓存
          return res
        }
      },
      cacheKey: (params)=>{
        // 使用参数生成唯一的key
        return "cache-key-xxx" + params.id
      }
    }
  }
]

说明

  1. name：service暴露出的方法名，如果 name="get123"，则调用时service.get123(...)
  2. handler：api中的正常请求方法
  3. cache：对 handler 结果是否缓存，默认为 false
  4. sync：handler 是否为同步，默认为 false
  5. transform：cache 为 true 时，对 handler 的返回结果做缓存处理，示例中对 res.success 进行判断是否缓存,返回值为 undefined 或 null 不缓存。
  6. cacheKey：cache 为 true 时，需要自定义缓存key，函数参数为 hander 参数，返回值为string类型

#api缓存

service 对象可对指定请求结果做缓存，方法中的部分参数会作为缓存key的一部分

注意

缓存会在页面刷新后清空

#资源缓存

缓存层会对 .mesh、 .mat、 .jpg、 .drs、 .json静态资源进行缓存。

#3. utils

#3.1. 鼠标事件监听

鼠标事件 source code
function useMouse(option) {
  let _target;
  if (option.target instanceof EventTarget) {
    _target = option.target;
  } else {
    if (document.querySelector(option.target) == null) {
      throw new Error("target is not HTMLElement");
    } else {
      _target = document.querySelector(option.target);
    }
  }
  function cb(e) {
    const o = {
      mouseLeft: e.buttons == 1,
      mouseRight: e.buttons == 2,
      mouseMiddle: e.buttons == 4,
      MouseEvent: e
    };
    option.onMove(o);
  }
  _target.addEventListener("mousemove", cb);
  return {
    clear: function() {
      _target.removeEventListener("mousemove", cb);
    }
  };
}
用法
const instance = useMouse({
  target: "#canvas",
  // 移动时执行此函数
  onMove(e) {
    console.log(e);
    // e = {
    //  mouseLeft: false,   移动时鼠标是否左键按下
    //  mouseRight: false,  移动时鼠标是否右键按下
    //  mouseMiddle: false, 移动时鼠标是否中键按下
    //  MouseEvent: {} 原生event事件
    //}
  }
})
// 移除移动监听事件
instance.clear()
