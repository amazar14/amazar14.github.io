#快速设计

#1. 平面布置

由于 平面布置页面 在快速设计中通过iframe嵌入，软件 与 平面布置页面 不能直接通信，所以增加 bridge 层做消息中转。
平面布置页面 与 软件 通信的业务数据和websocket方式保持一致。

#1.1 初始化

bridge需要从软件端获取 jsonUrl , tempUrl , spaceName。用来初始化平面布置

homedesign send
{
  type: 'message',
  from: 'homedesign',
  to: 'plane_layout'
  data: {
    code: "INIT_PARAMS"
    data: {
      jsonUrl: string,
      tempUrl: string,
      spaceName: string
    }
  }
}

#1.2 布局信息

web端需要从软件获取布局信息

web send
homedesign send
  {
    type: 'message',
    from: 'plane_layout',
    to: 'homedesign',
    data: {
      code: "GETLAYOUTPREVIEWIMG"
      data: {
        rooms: [],
        layouts: [],
        requestId: string
        fileid: string
      }
    }
  }
  {
    type: 'message',
    from: 'homedesign',
    to: 'plane_layout',
    data: {
      code: "LEGENDADJUSTBYRULE_RECEIVE",
      data: {
        fileid: ""
        hasEnterArea: null,
        layouts: [],
        rooms: []
        useJs: 1,
      }
    }
  }

#接入bridge

#1 iframe

// 注册到bridge中
window.parent.postMessage({
  type: 'regist',
  clientId: string // 见下表
}, '*')

// 发送消息
window.parent.postMessage({
  type: 'message',
  from: string, // 见下表
  to: string, // 见下表
  // 业务数据
  data: {
    code: string
    data: {}
  }
}, '*')

// 接收消息
window.addEventListener('message', function (ev) {
  const { type, from, to, data } = ev.data
  if (type === 'message' && to === clientId) {
    // do something
  }
})
